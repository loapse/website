Installation

Get set up in 60 seconds.

Bookmarklets

  1. 01
    Show your bookmarks bar

    Chrome/Firefox: Ctrl+Shift+B — Safari: View → Show Favourites Bar

  2. 02
    Drag the button

    Grab the "Drag to bar" button from any card and drop it onto your bookmarks bar.

  3. 03
    Or copy the code

    Hit the Copy button, right-click your bookmarks bar → Add bookmark, paste as the URL.

Userscripts

  1. 01
    Install a manager

    Download Tampermonkey or Violentmonkey.

  2. 02
    Create a new script

    Click the extension icon → Dashboard → + to open the editor.

  3. 03
    Paste and save

    Hit Copy script, paste into the editor, press Ctrl+S. Done.

About

Scriptshelf is a no-nonsense collection of browser utilities. No accounts, no tracking, no paywalls — just scripts that work. Every piece of code here is readable, commented, and short enough that you can understand exactly what it does before running it.
